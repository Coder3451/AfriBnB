# AfriBnB - Home Africa

## Description
This is the first step of the AfriBnB project. It implements a command interpreter to manage AfriBnB objects.

## Command Interpreter
### How to start
```bash
./console.py
```

### How to use
```text
- create <class name> - Create new instance
- show <class name> <id> - Show instance
- destroy <class name> <id> - Delete instance
- all [class name] - Show all instances
- update <class name> <id> <attr> <value> - Update attribute
- quit or EOF - Exit
```

### Examples
```text
(afribnb) create BaseModel
(afribnb) show BaseModel 1234-1234
(afribnb) all
(afribnb) quit
```
